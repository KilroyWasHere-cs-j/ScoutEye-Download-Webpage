ScoutEye Tracker

Getting started

Follow the download instructions located on the Downloads. After download perform the following.
ScoutEye is the data collection tool. This is the program that should be installed on each scouts computer.

Scouting levels

The ScoutEye Tracker has two discrete levels. Professional and Amateur. Since both Professional and Amateur levels can be configured separately from a configuration file. Please see Configuration for instruction. The different levels so that scouts can pick the level of scouting that is right for them.

Professional Level

As the name implies Professional level, is meant to be used by either dedicated scouts or scouts that want to collect the most complete match and team data. The lead scout should configure Professional with the intent to collect the most complete data.

Amateur Level

Amateur level is identical to Professional in reguards to functionality and UI layout. The difference is in the configuration, Amateur is intended to be used by no dedicated scouts and by scouts that do not feel the need to take as careful or detailed match data.

Configuring trackable parameters

As stated in Scouting Levels both levels Professional and Amateur can be customized by a configuration. Editor accessed by clicking the Configure ScoutEye in the home screen of ScoutEye.
